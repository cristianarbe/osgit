# osgit

osgit is a wrapper for apt-get that adds the capability of managing the list of installed packages as a git repository.

## Index

* [Getting started](#)

## Installation

1. Download the latest version,

```
$ git clone https://github.com/cristianarbe/osgit.git
```

2. install it

```
$ cd osgit
$ sh install
```

3. and done!
